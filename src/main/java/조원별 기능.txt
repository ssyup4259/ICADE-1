--------------------------------------------여정민------------------------------------------------------

- 구매 기능
  - 포인트 적립
  - 제품 재고 감소
  - 구매완료시 구매한 목록 출력

- 환불 기능
  - 포인트 감소
  - 제품 재고 증가
  - 환불시 받을 계좌 입력

- 결제 기능
  - 운송장 번호 생성
  -(카드결제 기능)

- 찜하기
  - 찜목록내에서 삭제
  - 삭제하기전까진 사라지지않음
  - 페이징 처리

- 최근본 상품 목록
  - 쿠키 사용
  - 10개정도만 보여지게 구현

--------------------------------------------조일준------------------------------------------------------

- 회원가입
  - 가입시 일반유저(admin, seller, customer) o
  - 이메일 인증 o
  - 아이디(한글입력방지,중복체크,영어숫자 혼용사용)o
  - 비밀번호(한글입력방지,일치여부확인,영어 숫자 특수문자 혼용사용,capslock켜져있는지 확인)o
  - 닉네임(중복검사)o
  - 이메일(중복검사, 이메일 형식이 맞는지 확인)o
  - 가입 양식 작성 못하면 회원가입 불가능

- 상품 상세 페이지
  - 일반(이미지 추가 가능)후기
  - 상품문의
  - 구매 버튼 만들기
  - 가능하다면 aJax로 탭 구현
  - 장바구니 버튼 만들기
  - 찜하기 버튼 만들기

- FAQ 게시판
  - (예시)https://www.cjlogistics.com/ko/support/faq - 자바스크립트 이용
